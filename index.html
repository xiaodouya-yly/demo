提示
  • 当单一文件较大或文件数量较多时，加载页面将需要更多时间。如果点击页面中的各项图标时不起作用，请稍等一小会再试
  • 由于 GitHub 的设置问题，此页面的文件名在左侧的文件列表中显示为 Index

功能说明

  • 点击右侧边栏右上角的 太阳/月亮 按钮，可以切换页面的亮暗模式
  • 在左侧边栏顶部的 搜索栏 中输入关键词，可以对所有文件进行全文搜索
    • 本地 HTML 文件尚未开启 搜索栏 功能
经豆芽测试，搜索栏 对中文的支持不够好，不建议搜索中文
  • 点击左侧边栏右上角和右侧边栏左上角的 隐藏/显示边栏 按钮，可以收起或展开左右边栏
    • 将鼠标悬停在两侧边栏的边线上，出现双向箭头后向左/右拖动可以调整边栏宽度
    • 调整边栏宽度后，将鼠标悬停在两侧边栏的边线上，出现双向箭头后双击可以恢复默认栏宽
左侧边栏收起后，边栏图标也会被隐藏。此时只需点击页面左上角的空白区域，即可再次展开左侧边栏（右侧边栏不存在这个 Bug）
  • 搜索栏 下方是一个可交互的文件双向链接 关系图谱，点击图谱中的节点会直接跳转到对应的文件
    • 点击 关系图谱 右上角的 箭头 或图谱外的任意空白区域，可以放大或缩小查看整个图谱
    • 点击 关系图谱 的圆形节点进行拖动，可以调整各个节点的位置
关系图谱 建议放大查看，因为一般来说文件标题都比较长，显示在小图中会全部堆叠到一起，导致文字无法看清
关系图谱 功能的 Bug
  • 在某些电脑上，图谱的节点和链接可能无法正常显示，原因未知
  • 偶尔图谱的节点会疯狂地到处跳动，此时在左侧文件列表中点选其他笔记切换一下页面即可（有时切换也没用，那就重新刷新一下页面吧 🥲）
  • 关系图谱 下方是 文件导航 ，可以通过点击文件名来切换当前展示的文件
    • 点击 文件导航 顶部 LIST OF PUBLICATION 右侧的上下双向 箭头，可以将文件列表全部展开或全部折叠
    • 在有文件夹的情况下，点击文件夹标题前的 箭头，可以展开或折叠下属的子文件（夹）列表
  • 右侧边栏的主体部分是文件的 大纲目录 ，可以通过点击章节标题来跳转到相应的内容
    • 点击 大纲目录 顶部 TABLE OF CONTENTS 右侧的上下双向 箭头，可以将文件大纲全部展开或全部折叠
    • 点击章节标题前的 箭头，可以展开或折叠下属的章节标题列表
较长的文件（夹）名称或标题名称无法在侧边栏中完整显示，此时需要手动调整边栏宽度，调整方式见上文
  • 如果有其他文件链接到正在打开的文件，右侧边栏底部会显示所有 链接当前页面的其他笔记 列表
