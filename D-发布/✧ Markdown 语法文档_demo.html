一、官方教程


二、标题

  • 在文本前添加 # 表示标题，# 必须在行首
  • # 的数量代表了标题的级别，共支持 6 级标题
Markdown 中凡是不成对的符号，后面都需要空格
  • 不同的 Markdown 应用程序处理 # 和标题间空格的方式不一致，为了不影响兼容性，最好用一个空格在 # 和标题之间进行分隔
在 MN4 中五级和六级标题的大小差距不大

三、分隔线

  • 在单独一行上使用三个或多个星号 (***)、破折号 (---) 或下划线 (___) ，并且不能包含其他内容
为了不影响兼容性，最好在分隔线前后都添加空白行

四、正文

4.1 强调

  • 粗体：在文本前后各添加两个星号，或使用快捷键 Cmd + B

  • 斜体：在文本前后各添加一个星号，或使用快捷键 Cmd + I

  • 粗体和斜体：在文本前后各添加三个星号或下划线

  • 高亮：在文本前后各添加两个等号、使用快捷键 Cmd + H或使用 <mark> 标签

    <mark>高亮</mark>

  • 实下划线或虚下划线：使用 <u> 标签或 <span> 标签的 style 属性

    <u>实下划线</u>
<span style="border-bottom:3px dashed blue">虚下划线</span>

  • 删除线：在文本前后各添加两个波浪线 ~

4.2 颜色

  • 使用 <font> 标签的 color 属性来修改文字颜色

    <font color="red">红色</font>
<font color="blue">蓝色</font>
<font color="rgb(200, 100, 100)">rgb颜色值</font>
<font color="#FF00BB">十六进制颜色</font>
<table><tr><td bgcolor=yellow>背景（黄色）</td></tr></table>

4.3 字号

  • 修改字号的三种方式：

    • <font> 标签 size 属性
    • <small> 或 <big> 标签
    • 修改 style 样式
    <font size="1">1号字 最小</font>
<font size="3">3号字 默认</font>
<font size="7">7号字 最大</font>
<small>小号字体</small>
<big>大号字体</big>

  • 字号与颜色的综合使用

    <font size="5" color="red">字号为5的红色文本</font>

4.4 字体

  • 使用 <font> 标签的 face 属性来修改字体

    <font face="HEI">黑体</font>
<font face="KAI">楷体</font>
<font face="GB18030 Bitmap">扩展字体</font>

4.5 上下标

  • 使用 <sup> 和 <sub> 标签来为字体添加上下标

    <sup>上标</sup>
<sub>下标</sub>

4.6 段落与换行

  • 使用 空白行 分隔文本来创建段落
严格的 Markdown 格式要求各个段落之间必须空行，避免使用空格（Spaces）或制表符（Tabs）来缩进段落
  • 段落内部的多个空格和段落之间的多个空行，在阅读视图中会被合并显示为单一空格或空行
  • 要想显示多个空格或空行，可以在笔记中使用 &nbsp;（空格）或 <br>（换行）
在 MN4 中换行即可标记新段落，使用键盘输入时按 Opt + Enter 换行
默认卡片编辑中的字体、字号等功能和 Markdown 输入模式不兼容，从前者转化为后者时，前者的编辑痕迹将消失，需要谨慎使用该功能

4.7 行对齐

  • 使用 <center> 标签和 <p> 标签的 left / right 属性实现行对齐

    <center>居中</center>
<p align="left">左对齐</p>
<p align="right">右对齐</p>

需要注释的文本[^1]

[^1]: 脚注内容 %% 脚注内容可以插入到文中的相应位置，阅读视图下仍然会被渲染到文末 %%

  • 块引用可以包含其他 Markdown 格式的元素

六、列表

6.1 无序列表

  • 在每个列表项前添加破折号 (-) 并紧跟一个 空格
  • 使用 Tab 键缩进一个或多个列表项可创建嵌套列表，每个 Tab 表示一层

6.2 有序列表

  • 在每个列表项前添加数字和英文句点，并紧跟一个 空格
  • 源码中的数字与渲染后并非一一对应，因此数字不必按顺序排列，但第一个数字决定了列表第一项的序号

6.3 任务列表

  • 在无序列表的破折号 (-) 和 空格 之后，用带 空格 的中括号 ([ ]) 来声明复选框
破折号与中括号、中括号与文本之间都需要空一格
  • 在括号中写入 x 或其他任意字符都可实现选中效果
    • Java
    • Python
可用 Tab / Shift + Tab 来缩进/取消缩进一个或多个已选择的列表项，以便快速组织列表

6.4 在列表中嵌套其他元素

  • 段落、引用块、图片：要在保留列表连续性的同时添加另一种元素，需将该元素缩进四个空格或一个制表符
  • 代码块：
    • 通常采用四个空格或一个制表符缩进（当它们被放在列表中时，采用八个空格或两个制表符缩进）
    • 围栏式代码块只需整体缩进一个制表符
所嵌套的元素与前后的列表文本之间必须各空一行
  • 列表：有序列表和无序列表之间可以互相嵌套

七、链接

  • 链接名称放在中括号内，链接地址放在圆括号内
    • [链接名称](链接地址)

7.1 给链接增加标题

  • 链接标题：当鼠标悬停在链接上时显示的文字
  • 链接标题可选，它与链接地址之间以空格分隔
    • [链接名称](链接地址 "链接标题")

7.2 带格式化的链接

  • 如需强调链接，在链接语法前后增加星号
  • 要将链接表示为代码，在方括号中添加反引号
不同 Markdown 应用程序处理 URL 中空格的方式不一样，为了不影响兼容性，最好用 %20 替代空格

八、图片

  • 添加图片：在方括号中输入图片名称，并在括号前添加感叹号 (!)。图片链接放在圆括号内，其后可选择增加图片标题
    • ![图片名称](图片地址 "图片标题")
可以通过直接拖动图片到笔记的特定位置来插入图片，相应的图片会被自动添加到附件文件夹中
  • 给图片增加链接：将图片的 Markdown 置入方括号中，然后将链接添加在圆括号中
    • [![沙漠中的岩石图片](/assets/img/shiprock.jpg "Shiprock")](https://markdown.com.cn)

九、代码

9.1 行内代码

  • 将单词或短语包裹在反引号 (`) 中

9.2 转义

9.2.1 反引号

  • 若要表示为代码的单词或短语中包含一个或多个反引号，则可以将其包裹在双反引号 (``) 中

9.2.2 反斜杠

  • 在语法符号前加入反斜杠 (\) 进行转义，可将其渲染为普通文本进行展示

9.3 代码块

  • 将代码的每一行缩进至少四个空格或一个制表符

    1

  • 围栏式代码块（fenced code blocks）不用缩进 

    ```markdown
此处是 Markdown 格式的围栏式代码块
```

  • 代码块支持语法高亮功能，可在代码的第一行（开头的三个反引号后）声明语言来使用特定的语法高亮
